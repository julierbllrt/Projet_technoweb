# Projet_technoweb

Utiliser GIT:
- se mettre dans le dossier du projet sur ton ordi ex:D:/Document/Cours/Technoweb/projet
- faire clic droit->git bash here

Dans la console qui s'est ouvert (gitbash):
Pour récuperer un ou plusieurs fichier qui sont sur github.com:
- git pull

Pour transferer des fichiers de ton ordi vers github.com:
- git add .
- git commit -m "le message qui dit les trucs que vous avez changer/se que vous voulez mettre à jour entre guillemet"
-git push
